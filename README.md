# DoggyDynamic -- Canine smart jacket
This innovative wearable device has been designed to capture every lively canine movement using an inertial measurement unit sensor. Featuring seamless acceleration and gyroscope data upload to Firebase Dataset via Wi-Fi, it employs machine learning algorithms to bring a 3d model of the dog to life. As well as visualizing movements in real-time, a scroll controller enables users to select from past dates and times on an interactive webpage.
